# Contributing

The canonical contribution guide lives at [../CONTRIBUTING.md](../CONTRIBUTING.md).

Use this short checklist before opening a pull request:

- Fork and clone the repository.
- Create a focused branch.
- Keep changes scoped.
- Add or update tests.
- Update documentation.
- Run `./mvnw clean verify`.
- Complete the pull request template.
- Do not include secrets, tokens, private logs, or sensitive evidence.
