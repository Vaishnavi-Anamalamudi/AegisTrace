# Roadmap

Roadmap items describe planned work and should not be presented as shipped behavior.

## v1.0.x

- Stabilize documentation and release process.
- Add controller authorization tests.
- Add screenshot and demo media from a clean local run.
- Add issue labels and release drafter configuration.

- Introduce OpenAPI documentation.

- Add scan status tracking.
- Add exportable incident reports.
- Add metrics for scan duration, event volume, and alert severity.
