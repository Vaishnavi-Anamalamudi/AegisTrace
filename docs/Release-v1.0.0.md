# Release v1.0.0

## Highlights

- Spring Boot SOC dashboard with operational entities.
- JWT authentication and role-based access control.
- Nmap-backed network scanning with TCP fallback.
- Scan history with search and CSV export.
- Security event, alert, incident, evidence, IOC, and threat-intel workflows.
- Docker Compose deployment.
- Maven CI workflow.
- Professional open-source documentation and governance files.
