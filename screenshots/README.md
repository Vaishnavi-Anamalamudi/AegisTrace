# Screenshot Gallery

Store redacted publication screenshots in this folder.

## Required Files

| Screen | Filename |
| --- | --- |
| Dashboard | `dashboard.png` |
| Login | `login.png` |
| Incident response | `incident.png` |
| Evidence vault | `evidence.png` |
| Scan history | `scan-history.png` |
| Settings | `settings.png` |
| Analytics | `analytics.png` |

## Capture Rules

- Use a clean local environment.
- Redact private IPs, hostnames, usernames, tokens, and evidence identifiers.
- Keep browser chrome out of final captures unless documenting a workflow.
- Prefer 1440px desktop width for README screenshots.
- Do not commit sensitive logs or production screenshots.
