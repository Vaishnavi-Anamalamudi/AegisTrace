# Backend

The backend application follows the standard Maven source layout under:

```text
src/main/java/com/vaishnavi/aegistrace
```

Primary backend boundaries:

- Controllers for API and page adapters
- DTOs for request and response contracts
- Services for application workflows
- Repositories for persistence
- Entities for JPA state
- Security for JWT and roles

See [../docs/DeveloperGuide.md](../docs/DeveloperGuide.md).
